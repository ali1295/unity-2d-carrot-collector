# 🎮 Carrot Collector (Unity 2D Game)

## 📌 Description
Carrot Collector is a 2D Unity game where the player collects carrots while avoiding obstacles and managing movement.

## 🚀 Features
- Player movement system
- Object collection system
- Score tracking
- Basic game mechanics

## 🛠 Built With
- Unity (C#)
