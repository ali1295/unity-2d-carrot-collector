# 🥕 Unity 2D Carrot Collector

A simple and fun **2D collectible game** built using **Unity Engine** and **C#**, where the player collects carrots while managing movement and avoiding simple gameplay challenges.

---

## 🎮 Gameplay Overview

In this game, the player controls a character in a 2D environment and collects carrots scattered across the map. The goal is to collect as many carrots as possible while enjoying smooth animations and responsive controls.

---

## 📌 Features

- 🥕 Simple 2D collectible mechanics  
- 🎮 Smooth player movement system  
- 🎯 Score-based gameplay  
- ✨ Basic animations and effects  
- 🌿 Interactive environment elements  
- 📱 Lightweight and beginner-friendly Unity project  

---

## 🛠️ Built With

- Unity Game Engine  
- C# Scripting  
- Unity 2D Physics  
- Sprite-based game design

## 🎮 Controls

| Action | Key |
|--------|-----|
| Move Left | A / Left Arrow |
| Move Right | D / Right Arrow |
| Move Up | W / Up Arrow |
| Move Down | S / Down Arrow |
